# ccc UI デザインガイド

## デザインコンセプト

**fzf ライクなターミナルネイティブ TUI**

- 検索が主役。起動した瞬間からタイプして検索できる
- メッセージ中心のナビゲーション（セッションは従）
- クロスセッション横断表示でブランチベースのフィルタリング
- ターミナルの ANSI 16色ベースで、どのテーマでも馴染む
- ボーダーは最小限。罫線のみで区切り

### 着想元
- **fzf**: 検索ファースト、結果リストのシンプルさ
- **lazygit**: ペイン分割、ステータスバーの情報設計
- **helix**: ミニマルなモード表示、ステータスライン

## カラーパレット（ANSI 16色ベース）

ターミナルのカラースキームに従う。固定色は使わない。

| 用途 | ANSI カラー | 備考 |
|------|------------|------|
| 検索バー文字 | Default (fg) | 通常のテキスト色 |
| 選択行背景 | DarkGray (bg) | シンプルな選択表示 |
| セッションセパレータ | DarkGray | 罫線とメタ情報 |
| Current branch | Default (fg) | 現在ブランチは通常色で目立たせる |
| 他のブランチ | DarkGray | 非アクティブなブランチ |
| User: ラベル | Blue + Bold | 発話者の区別 |
| Assistant: ラベル | Green + Bold | 発話者の区別 |
| コードブロック | Green | コード部分の識別 |
| 見出し | Cyan + Bold | マークダウン見出し |
| ステータスバー背景 | Cyan (bg) | 常時表示のヘルプ |
| マッチハイライト | Yellow + Bold | 検索一致箇所 |

## レイアウト

### 全体構成

```
40% 左ペイン                 60% 右ペイン
┌───────────────────────┬───────────────────────────────────┐
│ Search (2行)           │                                   │
├───────────────────────┤ Preview                           │
│ Messages (flex)        │ (全高さ使用)                       │
│                        │                                   │
├───────────────────────┤                                   │
│ Branches (動的高さ)     │                                   │
└───────────────────────┴───────────────────────────────────┘
ステータスバー (1行)
```

### 左ペイン内の分割

| セクション | 高さ | 説明 |
|-----------|------|------|
| Search | 2行固定 | 検索入力 + マッチ件数 |
| Messages | flex | クロスセッション横断メッセージリスト |
| Branches | 動的 (1-8行) | ブランチ一覧（`all` + 各ブランチ名） |

## コンポーネント

### 検索バー

- 常時表示、常時入力可能
- ヘッダーなし、パディング付き
- カーソル位置を `│` で表示
- 空の場合はプレースホルダー: `type to search...`（DarkGray）
- マッチ件数/総件数を右端に表示: `3/24`

### メッセージリスト（クロスセッション横断）

```
── 85449c0d main ──
> RustでHTTPサーバーを作りたい
  エラーハンドリングも追加して
  テスト書いて
── a1b2c3d4 feat/api ──
  REST APIのエンドポイント設計して
  認証ミドルウェア追加して
```

- セッションセパレータ: `── <session_id短縮> <branch> ──`
  - current_branch のブランチ名は fg 色、他は DarkGray
- 選択行: `>` プレフィックス + 反転表示
- 非選択行: 2スペースインデント
- 1行に収まるよう unicode-width で切り詰め + `…`
- user メッセージのみ表示
- ヘッダーなし

### ブランチペイン

```
▶ all
  main
  feat/api
  feat/ci
```

- 選択行: `▶` プレフィックス + Cyan 色
- current_branch は fg 色、他は DarkGray
- `all` は全セッション表示（デフォルト）

### プレビューペイン

**Full モード（デフォルト）:**
- マークダウンレンダリング（`render_markdown()` 使用）
- User: Blue + Bold / Assistant: Green + Bold
- ユーザーターン間はセパレータ（`────`）
- スクロール可能（Ctrl+d/u, gg/G）
- ヘッダーなし

**Short モード（Tab で切替）:**
- User/Assistant の1行ペア表示
- ペア間はセパレータ
- コンテンツは1行に切り詰め

### ステータスバー

```
 both  24 msgs │ C-j/k:msg  Opt-↑↓:branch  C-d/u:scroll  gg/G:top/btm  Tab:short/full  Enter:resume  C-y:copy
```

- 左端: モード状態（Cyan 背景）
- メッセージ件数
- キーバインドヘルプ（DarkGray）

## キーバインド

| キー | アクション |
|------|-----------|
| 文字入力 | 検索クエリに追加 |
| Backspace | 検索クエリ1文字削除 |
| Ctrl+l | 検索クエリクリア |
| Ctrl+j / Ctrl+k / ↑ / ↓ | メッセージ選択 上/下 |
| Alt+↑ / Alt+↓ | ブランチ選択 上/下 |
| Ctrl+d / Ctrl+u | プレビュー 半ページ下/上 |
| gg / G | プレビュー 先頭/末尾 |
| Tab | プレビューモード切替 (short/full) |
| Enter | セッション復元 (claude --resume) |
| Ctrl+y | セッションIDコピー |
| Esc / Ctrl+c / Ctrl+q | 終了 |

## トーン & ボイス

- ステータスメッセージは簡潔に: `Copied: abc123`
- エラーは原因を明示: `Failed to copy: clipboard unavailable`
- プレースホルダーは控えめ: `type to search...`
